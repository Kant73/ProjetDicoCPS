C version 0.4 char* style "itoa": Written by Lukás Chmela Released under GPLv3.

Utilisation d'une fonction C existante mais qui n'est plus présente avec GCC, peut être obselète... Permet de convertir un int "value" en une string "result" qui est la représentation du "value" en base "base" Non utilisée dans le projet mais utilisé surant la phase de developpement pour debugger certaine fonctions

Paramètres
valueL'entier dont on veut la representation dans la base "base"
resultLa chaine résulat qui va contenir la représentation de "value" dans la base "base"
baseLa base dans laquelle on veut obtenir la représentation de "value". "base" doit être compris entre 2 et 36.
Renvoie
L'entier "value" dans la base "base" sous la forme d'une chaîne de caractère
